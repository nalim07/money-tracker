# Changelog

All notable changes to this project will be documented in this file.

## [1.0.1] - 2026-05-06

### Fixed
- Horizontal scroll overflow on mobile devices.
- Transactions header and filter grid responsiveness.
- CSV import duplicates via new deduplication logic.
- Category mapping in CSV import (now correctly stores IDs).
- Automatic trimming of category names in CSV parsing.

### Added
- "Skipped" count indicator in Import CSV progress and summary.

## [1.0.0] - 2026-05-04

### Added
- CSV import functionality for transactions with wallet mapping.
- ConfirmDialog for reusable confirmation modals.

### Changed
- Updated app branding: logo to Money Tracker and replaced favicon.
- Enhanced category forms and settings layout.
